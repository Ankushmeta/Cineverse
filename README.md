# Cineverse

A simple static demo of a cinema booking UI (HTML/CSS/JS).

This repository was prepared from the user's provided `cineverse_app.html`.

## How to view

Open the `cineverse_app.html` file in a browser (double-click or use a simple static server).

Quick local preview using Python 3:

```bash
cd /workspace/cineverse
python -m http.server 8000
# then open http://localhost:8000/cineverse_app.html
```
